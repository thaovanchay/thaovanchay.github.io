Kính chào quý khách - Logoaz kính chúc quý khách sẽ ngày càng phát triển với nhận diện thương hiệu mới

--------------------------------------------------------------------------

Chúng tôi xin gửi đến quý khách hướng dẫn sử dụng
1. File .ai (phần mềm thiết kế illustrator) bao gồm 
- *CMYK là file gốc dùng cho việc in ấn, trong trường hợp quý khách muốn phát triển các ấn phẩm khác
- *RGB là file gốc dùng cho các việc thiết kế và ẩn phẩm sử dụng trên nền tảng Mobi, web

2. File ảnh
- .png là ảnh đã tách nền trắng, nghĩa là có thể kéo thả trên các bức ảnh mà không bị vướng nền
- .jpg là ảnh logo có nền trắng

3. File PSD
Là file logo chất lượng cao trên nền tảng Photoshop cho các bạn không dùng được phần mềm Illustrator

4. File Mockup: Là 2 file phối cảnh tặng kèm để quý khách có thể sử dụng để quảng bá thương hiệu mới

--------------------------------------------------------------------------
